# Team Profile Images

## Required Images

Place the following images in this folder:

1. **mohamed.jpg** - Mohamed Almaraghi's profile photo
2. **ali.jpg** - Ali Alsowaidi's profile photo  
3. **abdulrahman.jpg** - Abdulrahman Bumeajib's profile photo

## Image Guidelines

- **Format**: JPG or PNG
- **Size**: Recommended 300x300 pixels or larger (square format)
- **Quality**: High resolution for best display
- **File naming**: Use lowercase, match the filenames above exactly

## How to Add Images

1. Save your profile photos with the correct filenames
2. Copy them to this `images` folder
3. Commit and push to GitHub:
   ```bash
   git add images/
   git commit -m "Add team profile images"
   git push origin main
   ```

The website will automatically display your images in circular frames with a purple border.
